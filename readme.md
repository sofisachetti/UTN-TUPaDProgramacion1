# 💻 Programación I - Actividades Prácticas

Repositorio de actividades prácticas de la materia **Programación I**  
Tecnicatura Universitaria en Programación (modalidad a distancia) – 2026
UTN - Universidad Tecnológica Nacional

---

## 📌 Descripción

Este repositorio contiene ejercicios, trabajos prácticos y resoluciones desarrolladas durante la cursada de la materia.

El objetivo principal es aplicar los conceptos fundamentales de programación, desarrollando el pensamiento lógico y la resolución de problemas mediante algoritmos.

---

## 🎯 Objetivos de la materia

- Desarrollar pensamiento lógico y algorítmico  
- Resolver problemas mediante programación  
- Comprender y aplicar la sintaxis de un lenguaje (Python)  
- Implementar algoritmos correctos y eficientes  
- Fomentar buenas prácticas de desarrollo  

---

## 🧠 Contenidos

A lo largo de la cursada se trabajan los siguientes temas:

### 🔹 Módulo 1: Estructuras Secuenciales
- Variables, tipos de datos
- Operadores
- Entrada y salida de datos

### 🔹 Módulo 2: Condicionales
- if, elif, else
- Operadores lógicos y relacionales

### 🔹 Módulo 3: Bucles
- for
- while

### 🔹 Módulo 4: Trabajo Colaborativo
- Git y GitHub
- Control de versiones

### 🔹 Módulo 5: Listas
- Manipulación de colecciones de datos

### 🔹 Módulo 6: Funciones
- Modularidad
- Reutilización de código

### 🔹 Módulo 7: Estructuras de Datos
- Listas avanzadas
- Tuplas y diccionarios

### 🔹 Módulo 8: Manejo de Errores
- Excepciones (try / except)

### 🔹 Módulo 9: Archivos
- Lectura y escritura de archivos
- CSV

### 🔹 Módulo 10: Recursividad
- Funciones recursivas
